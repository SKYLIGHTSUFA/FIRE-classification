### План работы

:arrow_right: 80% изображений для обучение, остальные - для тестирования

:fast_forward: Изменить файлы `*.labels,` `*.name` `.*cfg`

:fast_forward: ~~*Скачать или*~~ скопировать данные в файлах в свои ___*.labels *.name .*cfg___ файлы
